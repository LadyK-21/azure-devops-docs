# Copilot Autofix for code scanning alerts (public preview)

Copilot automatically generates suggested code fixes for CodeQL code scanning alerts, complete with an explanation and a ready-to-apply diff — so developers can remediate vulnerabilities in-context and at scale without leaving Azure DevOps.
