# Copilot Code Reviews for Azure Repos (public preview)

Copilot Code Reviews for Azure Repos will go into public preview, bringing AI-assisted pull request reviews to Azure Repos. Copilot analyzes pull request changes and provides comments with potential issues and suggested improvements to help teams find problems earlier and improve code quality before merge.

## Summary of additional improvements

- Improvements to onboarding and enablement at the organization, project, and repository levels.
- Support for `copilot.instructions.md` and skills.
- Automatic code reviews for new pull requests.
- Ability to cancel an active Copilot code review run.
- Support for Managed DevOps Pools.
