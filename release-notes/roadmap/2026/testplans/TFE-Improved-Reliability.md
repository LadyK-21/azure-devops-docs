# Improve the Reliability and Performance of the Azure Test Plans' Browser Extensions

We are investing in significant back-end enhancements to the technology stack and frameworks powering by Azure Test Plans' Browser Extensions, specifically for Edge and Chrome browsers. These updates are designed to deliver even greater reliability and performance, ensuring a smoother experience for all users.
